# 🚀 Front end challenge

Essa é a minha resolução do [desafio frontend](https://github.com/nandokferrari/fernandev-react-challenge-03).

# 🖥️ Instalação

### Clone o repositório
```bash
git clone https://github.com/bmenegidio/fernandev-react-challenge-03
```

Instale as dependências

```bash
npm install
```

Inicie o frontend
```bash
npm run dev
```
